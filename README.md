#### Установка проекта

- установить версию python 3.7
- создать виртуальное окружение 
```shell script
python3.7 -m venv venv
```
- активировать виртуальное окружение
```shell script
source venv/bin/activate
```
- установить зависимости
```shell script
pip install -r requirements.txt
```
- запустить файл
```shell script
fem_solver_sfepy.py
